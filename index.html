개인정보처리방침

Coupang Partners (OpenClaw)
시행일: 2025년 1월 1일 | 최종 수정: 2025년 3월 6일

1. 개요

Coupang Partners (OpenClaw) (이하 "본 서비스")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 및 관련 법령을 준수하고 있습니다. 본 개인정보처리방침은 본 서비스가 어떤 정보를 수집하고, 어떻게 이용·보관·파기하는지에 대해 안내합니다.

2. 수집하는 개인정보 항목

본 서비스는 Instagram Graph API 및 YouTube Data API를 통해 다음과 같은 정보에 접근할 수 있습니다.

수집 항목 수집 목적 보유 기간
Instagram 비즈니스 계정 정보 (계정 ID, 사용자명) 콘텐츠 게시 및 계정 관리 서비스 이용 기간
Instagram 미디어 게시 권한 Reels/피드 자동 업로드 서비스 이용 기간
YouTube 채널 정보 (채널 ID) Shorts 영상 자동 업로드 서비스 이용 기간
게시 콘텐츠 성과 데이터 (조회수, 좋아요 수 등) 콘텐츠 분석 및 최적화 수집일로부터 1년

본 서비스는 이용자의 비밀번호, 결제 정보, 개인 메시지 등 민감한 정보를 수집하지 않습니다.

3. 개인정보의 이용 목적

수집된 정보는 다음의 목적으로만 사용됩니다.

  • Instagram Reels 및 YouTube Shorts 콘텐츠의 자동 게시
  • 게시된 콘텐츠의 성과 분석 및 전략 최적화
  • 서비스 운영 관련 기술적 문제 해결

4. 개인정보의 제3자 제공

본 서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우

5. 개인정보의 보관 및 파기

수집된 정보는 이용 목적이 달성된 후 지체 없이 파기합니다. 파기 절차 및 방법은 다음과 같습니다.

  • 전자 파일: 복구 불가능한 방법으로 영구 삭제
  • 보유 기간: 서비스 이용 종료 후 30일 이내 파기
  • 법적 보존 의무: 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관 후 파기

6. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 요구
  • 개인정보 수정·삭제 요구
  • 개인정보 처리 정지 요구
  • 서비스 연동 해제 및 데이터 삭제 요구

위 요청은 아래 연락처로 접수하시면 지체 없이 처리하겠습니다.

7. 사용자 데이터 삭제

이용자는 언제든지 본 서비스와의 연동을 해제하고 저장된 데이터의 삭제를 요청할 수 있습니다.

삭제 요청 방법

  • 아래 연락처 이메일로 "데이터 삭제 요청" 제목으로 메일 발송
  • Facebook 설정 → 앱 및 웹사이트 → Coupang Partners(OpenClaw) → 연결 해제

삭제 요청 접수 후 30일 이내에 모든 관련 데이터를 완전히 삭제하며, 삭제 완료 시 이메일로 통보합니다.

8. 쿠키 및 자동 수집 장치

본 서비스는 웹 기반 서비스가 아닌 API 기반 자동화 도구로서, 쿠키 또는 자동 수집 장치를 사용하지 않습니다.

9. 개인정보의 안전성 확보 조치

본 서비스는 개인정보의 안전성 확보를 위해 다음의 조치를 취하고 있습니다.

  • 접근 권한의 제한: 관리자 계정만 데이터에 접근 가능
  • API 토큰 암호화 저장
  • 전송 구간 SSL/TLS 암호화 적용

10. 개인정보처리방침 변경

본 방침이 변경되는 경우, 변경 사항은 본 페이지에 즉시 게시되며, 변경 전·후를 비교하여 확인할 수 있도록 합니다.
